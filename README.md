# Modelagem de Banco de Dados — Sistema de Imobiliária

Este repositório contém a modelagem completa do banco de dados para um sistema de gestão imobiliária, incluindo:

- Modelo Conceitual (DER)
- Modelo Lógico
- Modelo Físico (SQL)

## Estrutura

- /01-conceitual → DER no BRModelo
- /02-logico → Modelo lógico completo
- /03-fisico → Scripts SQL e Workbench

## Tecnologias Utilizadas

- BRModelo 3.0
- MySQL Workbench
- MySQL 8+
